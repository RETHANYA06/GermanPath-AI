# 🇩🇪 GermanPath AI

An AI-powered German learning platform designed to help learners master German from **A1 to B1** through interactive lessons, quizzes, reading, listening, grammar practice, and AI-assisted learning.

---

## ✨ Features

* 📚 Vocabulary Learning
* 🃏 Interactive Flashcards
* 🇩🇪 Grammar Lessons
* 📝 Grammar Quiz
* ⚡ Quick Quiz
* 📘 Practice Test
* 🎓 Mock Exam
* 📖 Reading Practice
* 🎧 Listening Practice (Audio)
* 🤖 AI German Tutor (Gemini AI)
* 📊 Progress Dashboard
* 🏆 Goethe Exam Preparation

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Google Gemini API
* gTTS
* HTML & CSS
* JSON
