🇩🇪 GermanPath AI

An AI-powered German learning assistant designed to help users learn German through translation, practice, and intelligent language interaction.

🚀 Overview

GermanPath AI is a smart language learning system focused on improving German skills through:

🇩🇪 German language learning support
🔄 English ↔ German translation
🧠 AI-powered conversation practice
📖 Vocabulary building assistance
✍️ Sentence correction and improvement
✨ Features
💬 Chat-based German learning assistant
🔁 Real-time translation (English ↔ German)
📚 Vocabulary and sentence suggestions
🧠 AI explanations for grammar usage
🗣️ Practice mode for daily German conversation
📊 Learning progress tracking (if implemented)
🏗️ Tech Stack
Python 🐍
Google Gemini API (Generative AI)
Flask / FastAPI (if backend used)
HTML, CSS, JavaScript (frontend if applicable)
dotenv for API key management
